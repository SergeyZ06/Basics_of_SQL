БД для CRM-системы финансовой коммерческой организации - банка.

1. Назначение БД:
	- хранение данных о клиентах банка (паспортные данные, персональная информация, финансовая информация);
	- хранение данных о банке (информация об услугах и предложениях банка, информация для обслуживания действующих счетов);
	- реализация инструментария для аналитики (наиболее успешные продукты банка*, рейтинг клиентов, разбиентие клиентов на группы*).
	
	* - не реализовано на момент 2018/04/18.

2. Структура БД:
	- таблица "clients_profiles" содержит базовую информацию об учётных записях клиентов, клиент может не обладать учётной записью и обращаться в отделения банка с паспотом;
	- таблица "clients_passports" содержит паспортные данные клиентов, данные обязательны к заполнению;
	- таблица "client_contacts" содержит контактные данные клиентов (номера телефонов, электронной почты, фактического адреса проживания, данные не обязательны к заполнению);
	- таблица "client_social_networks" содержит информацию о социальных сетях клиента;
	- таблица "clients_sessions" содержит информацию об обращениях клиента в банк (посредством мобильного приложения, звонка в отделение банка, посещения сайта, офиса или банкомата);
	- таблица "clients_operations" содержит информацию об операциях клиента (перевод средств с счета на счет и т.д.), операции привязаны к сессиям;
	- таблица "clients_resume" содержит персональную информацию клиентов для аналитики;
	- таблица "offers_credit" содержит информацию о кредитных предложениях банка;
	- таблица "offers_debit" содержит информацию о дебетовых предложениях банка;
	- таблица "accounts_credit" содержит информацию для обслуживания действующих кредитных счетов клиентов, процентная ставка корректируется для каждого клиента в соответствии с его рейтингом;
	- таблица "accounts_credit_status" содержит информацию о статусе кредитных счетов;
	- таблица "accounts_debit" содержит информацию для обслуживания действующих дебетовых счетов клиентов, процентная ставка корректируется для каждого клиента в соответствии с его рейтингом;
	- таблица "accounts_debit_status" содержит информацию о статусе дебетовых счетов.
